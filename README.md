
# fastapi-todo-backend

A high-performance asynchronous TODO API backend built with FastAPI and SQLite. This project demonstrates building a robust and scalable RESTful API with modern Python web frameworks.

## Features
- Create, Read, Update, Delete (CRUD) operations for TODO items.
- Asynchronous API endpoints.
- SQLite database integration.

## Technologies
- Python
- FastAPI
- Pydantic
- SQLite

## Setup
1. Clone the repository:
   `git clone https://github.com/yihang880/fastapi-todo-backend.git`
2. Install dependencies:
   `pip install fastapi uvicorn pydantic`

## Usage
Run the application:
```bash
uvicorn main:app --reload
```
Access the API documentation at `http://127.0.0.1:8000/docs`.
